# modbusey
A quick and dirty Modbus TCP client written in Python.

**Supported Functions:**
- `read_coils`
- `read_discrete_inputs`
- `read_holding_registers`
- `read_input_registers`
- `write_coil`
- `write_coils`
- `write_register`
- `mask_write_register`

**Requirements:**
- `pymodbus`
- `readline`
- `colorama`
